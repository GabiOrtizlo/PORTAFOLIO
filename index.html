Noti-World

Noti-World es una página web de noticias diseñada para ofrecer información actualizada y accesible de manera dinámica. El proyecto está enfocado en reducir la necesidad de crear múltiples archivos HTML al permitir la carga automática de noticias desde un archivo JSON o una base de datos. Esto facilita la gestión y actualización de contenido sin la intervención manual constante. El diseño web es totalmente responsive, inspirado en el estilo de periódicos como El Heraldo, con artículos alineados horizontalmente en pantallas de PC y apilados verticalmente en dispositivos móviles. El sistema de noticias se integra con la News API para obtener artículos en tiempo real, lo que mantiene a los usuarios informados sobre los últimos acontecimientos a nivel global.

Lenguajes y tecnologías utilizadas:

HTML

CSS

JavaScript

JSON

API de News

Modern Recipe

Este proyecto es una aplicación web que utiliza la API de Spoonacular para mostrar recetas aleatorias. La aplicación se centra en brindar a los usuarios una variedad de recetas para explorar, y aunque está diseñada para estar en inglés, puede adaptarse fácilmente a otros idiomas.

Funcionalidades Principales

Carga de Recetas Aleatorias:

-La aplicación hace una solicitud a la API de Spoonacular para obtener un número específico de recetas aleatorias.

-Se utiliza la clave de API para autenticar la solicitud y obtener datos de recetas.

Presentación de Recetas:

Las recetas se muestran en tarjetas que incluyen:

Imagen: Una visualización atractiva de cada receta.

Título: El nombre de la receta.

Descripción: Un breve resumen de la receta, que incluye ingredientes y preparaciones.

Interactividad:

Al hacer clic en el título de la receta, los usuarios son redirigidos a la página original de la receta en Spoonacular para más detalles.

Tecnología Utilizada

-HTML: Para estructurar el contenido de la página.

-CSS: Para el estilo y la presentación visual de la página.

-JavaScript: Para la lógica de carga de recetas y la manipulación del DOM.

API de Spoonacular: Para obtener datos de recetas.

Legends of Marvel

Legends of Marvel es una emocionante página web creada para los fanáticos del universo Marvel. Este proyecto utiliza la API de Marvel, lo que permite a los usuarios explorar una amplia gama de personajes, series y cómics de Marvel en tiempo real. La plataforma ofrece una interfaz intuitiva y atractiva, facilitando la navegación y la búsqueda de información sobre los héroes y villanos favoritos de los usuarios.

Tecnologías Utilizadas:

HTML: Para la estructura y el contenido de la página.

CSS: Para el diseño y el estilo visual, asegurando que la experiencia del usuario sea atractiva y responsiva.

JavaScript: Para la interactividad y la conexión con la API de Marvel, permitiendo una experiencia dinámica y actualizada.

API de Marvel: Para obtener datos.
